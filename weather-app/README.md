# Weather App (Java + JSP + Tomcat)

A starter medium-level Java web app using Servlets + JSP that fetches weather from Open-Meteo (no API key).

## Tech
- Java 21
- Tomcat 10+
- Maven
- HTML/CSS
- Open-Meteo API

## How to Run
1. Open project in IntelliJ IDEA
2. Configure Tomcat 10+ as Run Configuration
3. Build with Maven
4. Run on Tomcat
5. Open http://localhost:8080/weather-app/
